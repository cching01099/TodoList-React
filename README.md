# Todo List X MVC with React

todoList 實作功能
編輯、刪除、狀態切換

### 請遵循以下步驟

#### step 1

打開終端機 輸入指令
`git clone  'https://github.com/cching01099/TodoList-React'`

#### step 2

輸入指令 `cd TodoList-React`
進入專案資料夾

#### step 3

輸入指令 `npm install`
安裝所需套件

#### step 4

輸入指令 `npm start`
啟動專案

#### step 5

啟動成功
網頁將在本地端
`http://localhost:3000/todo`運行

### 環境建構

- react 18.2.0
- Node.js 14.18.1
- npm 6.4.15
